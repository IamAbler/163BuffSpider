# 163BuffSpider

## 运行要求
1.Python3.X

## 使用步骤
1.首次运行前以管理员身份运行Install.bat  
2.获取Buff商品id与cookie  
3.运行main.py  
4.填入所需参数 game默认参数为csgo  
Tips：请不要将SleepTime参数设置过小，否则会被api接口屏蔽。

## 警告
Buff有反爬机制，请不要滥用本爬虫，尽量调大爬取间隔，以避免账号受到封禁。  
练手项目，仅做学习交流用途。
